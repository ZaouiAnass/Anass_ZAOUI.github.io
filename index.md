## Projects I've Built

### Swing Tracker — IoT Sports Analysis System
*Python, C++, Arduino*

This one was fun. I built an embedded system using Arduino with accelerometer and gyroscope sensors to capture real-time movement data. The interesting part was developing the signal processing algorithms in Python to calculate force, velocity, and swing angle from raw sensor data. I also created a mobile interface so athletes could see their performance metrics instantly. It's the kind of project where hardware meets software meets real-world application.

---

### Handwritten Digit Recognition
*Python, TensorFlow, CNN*

A deep learning project where I trained a Convolutional Neural Network on the MNIST dataset. Beyond just getting it to work, I spent a lot of time optimizing hyperparameters — learning rate, layer architecture, regularization — to push accuracy higher while fighting overfitting. The model ended up integrated into an automated testing pipeline. This project really solidified my understanding of how neural networks learn.

---

### Budget Tracker — Fullstack Web App
*React, Vite, Node.js, Express, MongoDB*

A complete personal finance management application built with the MERN stack. I designed the REST API from scratch with Node.js and Express, set up MongoDB for data persistence, and built a responsive React frontend with Vite. The app includes secure authentication, and I learned a ton about the full development lifecycle — from database schema design to deployment considerations.
